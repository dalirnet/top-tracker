# Top tracker

JavaScript client for **[Toptal](https://www.toptal.com/)** [tracker](https://www.toptal.com/tracker)

#### Install

```bash
npm install top-tracker
```

#### Import

```js
import topTracker from 'top-tracker'
```

#### Sample

```js
import {  } from 'top-tracker'

const parameters = {}
const input = {}

new (parameters)
    .call(input)
    .then((output) => {})
    .catch((error) => {})
```
